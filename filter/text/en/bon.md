Prints text from a template.

The {{aka}} command prints the text from a selected template.

Templates for creation of a new 'bonzai.Cmd':

* `quick` will print only the most commonly used 'bonzai.Cmd' fields.
* `full` will print all available fields.

Templates for other use-cases:

* `branchreadme` will print the standard README template for new Bonzai branch repositories.
* `gobadges` will print standard Go Badges.
